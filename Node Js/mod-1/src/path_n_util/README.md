# Node.js Path Module -- Detailed Guide

This README provides a detailed explanation of the Node.js **Path

## üöÄ Introduction

The **Path module** is a core Node.js utility that helps you work with
file and directory paths in a consistent and cross‚Äëplatform way
(Windows, Linux, macOS).

Import it using:

``` js
const path = require("path");
```

------------------------------------------------------------------------

## ‚≠ê Key Methods

### 1. `path.join()`

Joins path segments safely.

``` js
path.join("folder", "sub", "file.txt");
```

------------------------------------------------------------------------

### 2. `path.resolve()`

Returns an absolute path.

``` js
path.resolve("folder", "file.txt");
```

------------------------------------------------------------------------

### 3. `path.basename()`

Returns the filename.

``` js
path.basename("/home/user/file.txt");
```

------------------------------------------------------------------------

### 4. `path.dirname()`

Returns the directory name.

``` js
path.dirname("/home/user/file.txt");
```

------------------------------------------------------------------------

### 5. `path.extname()`

Returns the file extension.

``` js
path.extname("data.json");
```

------------------------------------------------------------------------

### 6. `path.parse()`

Breaks a path into useful parts.

``` js
path.parse("/home/user/app.js");
```

------------------------------------------------------------------------

### 7. `path.format()`

Builds a full path from an object.

``` js
path.format({ dir: "/folder", base: "file.txt" });
```

------------------------------------------------------------------------

### 8. `path.normalize()`

Fixes messy paths.

``` js
path.normalize("/folder//sub/../file.txt");
```

------------------------------------------------------------------------

### 9. `path.isAbsolute()`

Checks if a path is absolute.

``` js
path.isAbsolute("/home");
```

------------------------------------------------------------------------

### üîß Additional Utility

-   Path separator: `path.sep`
-   Current file: `path.basename(__filename)`
-   Current dir: `path.dirname(__filename)`
-   Build project path:

``` js
path.join(__dirname, "data", "users.json");
```
