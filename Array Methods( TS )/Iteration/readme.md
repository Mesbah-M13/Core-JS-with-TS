## Here are the most useful JavaScript array methods:

## Iteration

```bash
•  forEach(callback) - Execute function for each element
•  map(callback) - Transform each element, return new array
•  filter(callback) - Keep elements that pass test
•  reduce(callback, initial) - Reduce array to single value
```

## Search

```bash
•  find(callback) - Return first matching element
•  findIndex(callback) - Return index of first match
•  includes(value) - Check if value exists
•  indexOf(value) - Get index of value (-1 if not found)
```

## Modification

```bash
•  push(item) - Add to end
•  pop() - Remove from end
•  shift() - Remove from start
•  unshift(item) - Add to start
•  splice(start, deleteCount, ...items) - Add/remove at any position
•  slice(start, end) - Extract portion (doesn't modify original)
```

## Combination

```bash
•  concat(...arrays) - Merge arrays
•  join(separator) - Convert to string
```

## Ordering

```bash
•  sort(compareFn) - Sort in place
•  reverse() - Reverse in place
```

## Testing

```bash
•  some(callback) - Check if at least one passes test
•  every(callback) - Check if all pass test
```

## Modern additions

```bash
•  flat(depth) - Flatten nested arrays
•  flatMap(callback) - Map then flatten
•  at(index) - Get element (supports negative indices)
```
